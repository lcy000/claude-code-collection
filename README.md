# Claude Code Collection

由 CYouuu 整理，收錄 Claude Code 相關的自製功能、插件設定與 Skills 推薦，供個人使用與參考分享。

---

## 內容

### [Hooks — macOS 提醒通知](hooks/macos-notification.md)
自製的兩個 hooks，偵測使用者是否正在看著 Claude Code，分別在完成與需要授權時發出通知。

### [插件 — claude-hud 狀態列](plugins/claude-hud.md)
claude-hud 插件的安裝方式與個人設定，在終端機下方顯示即時狀態列。

### [Skills 推薦收藏](skills/collection.md)
值得安裝的 Claude Code Skills 整理，附說明與適用情境。
